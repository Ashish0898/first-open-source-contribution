# 🌟 Very First Open Source Contribution

Welcome! This repository is specially created for beginners to make their first open source contribution.

---

## 🚀 How to Contribute

1. Fork this repository
2. Clone your fork

   ```
   git clone https://github.com/YOUR-USERNAME/veryfirstopensourcecontribution3.git
   ```
3. Create a new branch

   ```
   git checkout -b add-your-name
   ```
4. Add your name in `contributors.md`
5. Commit your changes

   ```
   git commit -m "Added my name"
   ```
6. Push to GitHub

   ```
   git push origin add-your-name
   ```
7. Create a Pull Request

---

## 📌 Contribution Rules

* No spam PRs ❌
* Make meaningful contributions ✅
* Follow instructions properly
